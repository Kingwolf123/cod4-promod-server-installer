COD4 Promod Server Package
==========================

This package contains the install and startup layer for a Promod FPS 285 server.

What is included:
- setup scripts
- startup scripts
- template server_match.cfg
- folder stubs for base game files, Promod files, and custom maps

What is not included:
- base Call of Duty 4 game files
- COD4X server binaries
- Promod mod assets for redistribution
- custom map assets

Start here:
- Read BEGINNER_GUIDE.txt
- Run install_server.bat after you have copied the required game and mod files into the correct folders



If any issues or improvements, please open a issue or pull request on GitHub.
